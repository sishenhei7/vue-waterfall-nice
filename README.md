# vue-waterfall-nice

> vue 的瀑布流组件，支持无图、多图模式，支持自定义布局，支持服务端渲染，支持懒加载，使用非常简单！

本插件极大地借鉴了 [vue-waterfall-adaptive](https://github.com/geng452654716/vue-waterfall)

## Install

```
npm i vue-waterfall-nice -s
```
